# ObservAção

Sistema de gestão de solicitações e denúncias cidadãs, desenvolvido como projeto integrador do curso de Engenharia de Software — UniCesumar 2026.

---

## Sobre o projeto

O ObservAção permite que cidadãos registrem solicitações de serviços públicos e denúncias anônimas, acompanhem o andamento pelo número de protocolo, e que servidores/gestores tratem as demandas com rastreabilidade e organização. O projeto é alinhado aos ODS 16 (Paz, Justiça e Instituições Eficazes) e ODS 10 (Redução das Desigualdades).

---

## Tecnologias utilizadas

- Java + Spring Boot
- HTML, CSS e JavaScript
- SonarQube (análise de qualidade)

---

## Como executar

*Pré-requisitos:* Java 17+, Maven

bash
# Clone o repositório
git clone https://github.com/PauloMalavasi/Aep-Parte-2.git

# Acesse a pasta do projeto
cd Aep-Parte-2/aep

# Execute o projeto
mvn spring-boot:run


Acesse no navegador: http://localhost:8080

---

## Telas do sistema

| Tela | Descrição |
|------|-----------|
| Página Inicial | Hub de navegação com acesso a todas as funcionalidades |
| Nova Solicitação | Registro de solicitação de serviço público |
| Consultar Protocolo | Acompanhamento de solicitação pelo número de protocolo |
| Denúncia Anônima | Registro de denúncia sem identificação do cidadão |
| Painel do Gestor | Gestão e atualização de demandas (acesso restrito) |
